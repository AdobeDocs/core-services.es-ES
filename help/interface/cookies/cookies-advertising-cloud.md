# Cookies de Advertising Cloud{#advertising-cloud-cookies}

Advertising Cloud usa cookies para asignar eventos de participación de publicidad a eventos de conversión y, potencialmente, para utilizar esa información para optimizar las ofertas de publicidad.

  </tr> 
  <tr> 
   <td colname="col1"> <p>Ubicación </p> </td> 
   <td colname="col2"> <p>Dominio de nivel superior del anunciante (como example.com) </p> </td>

  <tr>

   <td colname="col2"> <p>Variable, pero normalmente 50 a 150 bytes </p> </td> 
  </tr>
